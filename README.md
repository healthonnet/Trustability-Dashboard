HON-Gold-Standard-Trustability
==============================

Gold standard benchmark dashboard

Deploy dashboard to github pages
--------------------------------

### Install

```code
$ npm install
```

Fetch `ghp-iport`:

```code
$ pip install ghp-import
```

### Build

```code
$ npm run build-pages
```

### Deploy

```code
$ npm run deploy-pages
```
